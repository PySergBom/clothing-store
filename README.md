### Modern store 
> Modern store e-commerce with Django framework


Some of the Features 

- Categorization of Products
- Caching (Redis)
- Multiple Signup Ways (OAuth (Github), Email & Password)
- Email verification (Celery, Yandex)
- Basket
- Orders
- Pagination
- Stripe Paying System
